# 会议氛围组 MCP 服务

会议氛围组 MCP 是一个专门用于改善会议和团队活动氛围的工具集合。该服务提供了多种功能，包括生成破冰游戏和提供轻松笑话，旨在帮助团队建立更好的沟通氛围和提升团队凝聚力。

## 功能特性

### 破冰游戏生成器
根据参与人数智能生成适合的破冰游戏，帮助团队成员快速熟悉彼此，打破初次见面的尴尬氛围。

### 笑话提供器
提供精选的轻松笑话，为会议或团队活动增添欢乐气氛。

## 技术架构

本服务基于 FastMCP 框架构建，使用 Python 语言开发。通过集成大语言模型，能够智能生成符合需求的内容。

## 使用方法

### 方式一：使用 uv 运行（推荐）

1. 安装依赖:
```bash
uv sync
```

2. 运行服务:
```bash
uv run python server.py
```

### 方式二：使用 npx 快速部署（需要 Node.js）

```bash
npx meeting-atmosphere-mcp
```

服务默认使用 SSE 传输模式，在 8080 端口运行。
